# ⚡ Diseño Electrónico y PCB - LANCEA

Este directorio contiene los archivos fuente del diseño de la PCB para la aviónica del proyecto.

## 🛠 Especificaciones
* **Software:** KiCad 9.0
* **Capas:** 2 capas
* **Tecnología:** Componentes Through-hole (THT) para mayor robustez mecánica.

## 📂 Estructura de Archivos
* **Esquemático:** `.kicad_sch` (Lógica del circuito).
* **PCB Layout:** `.kicad_pcb` (Diseño físico de la placa).
* **Proyecto:** `.kicad_pro` (Archivo maestro).
* **Librerías y 3D:** Carpetas de símbolos, huellas y modelos STEP.

## 📥 ¿Cómo descargar los archivos?

Si necesitas descargar un archivo individual (como el esquemático o la PCB) directamente desde el navegador:

1. Haz clic sobre el archivo que deseas (ej. `.kicad_pcb`).
2. En la esquina superior derecha de la vista del archivo, haz clic en los **tres puntos (`...`)**.
3. Selecciona la opción **Download**.
4. **Atajo de teclado:** También puedes presionar `Ctrl` + `Shift` + `S` mientras visualizas el código del archivo para guardarlo automáticamente.

> **Nota:** Se recomienda descargar todo el repositorio como un `.zip` desde la página principal para mantener la integridad de las librerías y modelos 3D.
